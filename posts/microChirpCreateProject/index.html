Abstract

infineon 60 Ghz Radar SDK Guide, 现在手中只有infineonMCU760Ghz雷达Demo板, 所以, 按照现有方案进行开发.

MCU7 + sensor-xensiv-bgt60trxx

主控MCU: atmel ATSAMS70Q21

Atmel 被MicroChip 在MicroChip 中现在是asm系列, 两条产品线互为补充

主要分为两部分 - 设置microChip工程创建 - infineon雷达配置

Referance

infineon 60Ghz Radar sdk : https://github.com/Infineon/sensor-xensiv-bgt60trxx

MicroChip Studio EXE: https://atmega32-avr.com/free-download-atmel-studio-7-latest-version-for-windows-macos/

microChip use Guide: https://onlinedocs.microchip.com/pr/GUID-4E095027-601A-4343-844F-2034603B4C9C-en-US-4/index.html?GUID-C3B1E8DA-FFC9-4DBB-9AB6-0C9C5214BB4A

MPLAB download : https://www.microchip.com/en-us/tools-resources/develop/mplab-x-ide#tabs

install MC ide demo: https://blog.csdn.net/ba_wang_mao/article/details/123633594

安装Microchip Studio

普通的工程创建( 不推荐 )

项目创建的很简单, 适合小型项目

工程创建( 推荐 )

进入到atmel工程创建地址, 搜索对应的芯片, 选择后进入相关配置.

对项目具体时钟, 引脚进行配置

工程配置导出

将当前的芯片配置进行导出成工程文件, 根据所需项目进行导出.

点击Download之后会看到一个.atzip文件

ide导入配置完成的工程文件

打开ide, 拖拉放入ide界面,弹出配置, 点击ok即可

Build

编译项目

编译完成, 到此项目创建完成, 下面进行雷达的相关配置.
