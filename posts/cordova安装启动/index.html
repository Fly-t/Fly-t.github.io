1、创建应用程序

参考： https://www.w3cschool.cn/cordova/cordova_first_application.html

cordova create CordovaProject io.cordova.hellocordova CordovaApp

  • CordovaProject 是创建应用程序的目录名称。

  • io.cordova.hellocordova 是默认的反向域值。 如果可能，您应该使用您自己的域值。

  • CordovaApp 是您应用的标题。

2 、 添加平台

进入到创建的项目（CordovaProject）目录中

cordova platform add android

### 3、构建和运行 安装javaSDK 下载地址：https://www.oracle.com/java/technologies/downloads/#jdk17-windows 

JAVA_HOME  = 安装的jdk位置
%JAVA_HOME%\bin;
%JAVA_HOME%\jre\bin

#### 测试jdk是否安装成功

java -version

### 安装安卓SDK installer_r24.4.1-windows.exe 下载地址 https://www.androiddevtools.cn/

#### 添加环境变量

ANDROID_HOME（不推荐）
ANDROID_SDK_ROOT（推荐）
D:\environment\Android\android-sdk
%ANDROID_SDK_ROOT%\tools
%ANDROID_SDK_ROOT%\platform-tools

%ANDROID_HOME%\tools
%ANDROID_HOME%\platform-tools

测试SDK是否安装成功 adb --version ### 接下来下载gradle要不然出现一下报错

Using Android SDK: D:\environment\Android\android-sdk
Could not find an installed version of Gradle either in Android Studio,
or on your system to install the gradle wrapper. Please include gradle
in your path, or install Android Studio

gradle：下载地址https://services.gradle.org/distributions/

gradle添加环境变量

### 3、运行

cordova build android

外部设备启动：（内部的报错，懒得搞了，真实机器测试）

cordova run android
